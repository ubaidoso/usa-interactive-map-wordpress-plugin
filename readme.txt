# USA Interactive Map WordPress Plugin

A custom WordPress plugin that displays an interactive USA map with multiple pins per state. This plugin allows adding dynamic state markers (SC, GA, NC, etc.) from the WordPress admin panel.

## 🔥 Features

- Multiple pins in the same state
- Custom state marker colors
- Admin panel to manage locations
- Responsive SVG USA map
- Lightweight and fast loading
- Easy WordPress integration

## 🛠 Technologies Used

- JavaScript
- SVG Map
- WordPress Plugin Development
- PHP
- CSS

## 📍 Supported States Example

- South Carolina (SC)
- Georgia (GA)
- North Carolina (NC)

## 🚀 Installation

1. Download the plugin
2. Upload to `/wp-content/plugins/`
3. Activate from WordPress admin
4. Add your state pins from the plugin settings
